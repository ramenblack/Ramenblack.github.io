CSS

Styling websites has always been just as important, if not more so than developing content for said website. Good web design attracts users, and draws attention to the web content.

Project Timeline

My Web Design Journey.

  • September 2018

    My Humble Beginnings

    My first Webpage, simply titled 'localhost', was very simple, featuring no css or style, and consisting of nothing more than a few simple links and some text. Definitely nothing to be proud of, however we all have to start somewhere, and this provided the opprotunity for me to dip my feet into HTML. You can visit it here.

  • November 2018

    Keegy.ioe

    building upon 'localhost', and applying tutorials from W3Schools, I began work on my second website, developing it both during class, and at home. This website debuted my use of external style sheets, making good use of w3.css. This website is still in development today, and features a php login page, which allows users to build a profile page. You can visit it here.

  • March 2018

    Skanoodle.ioe

    Skanoodle.ioe is the third website I've worked on, and was a joint effort between me and my group, consiting of Cynthia, Carlos, and Micah. Admittedly, Cynthia had the most influence over the website, however, I contributed the logo design, php, and helped on various styling elements. Skanoodle is a fictional bank, and consists of a contact page, with working comments, and a modern, yet friendly design. You can visit Skanoodle here.

  • April 2018

    Project Brisbou

    The latest installment to my portoflio of websites, Project Brisbou was developed entirely at home, and deployed to an apache web server that I set up on my laptop. Brisbou is a search engine mock up that I made to practice my html, photoshop, and web design skills. This is also the first website I've made that uses boostrap other than W3.css, as Brisbou runs off of Landing Sumo. You can view it here.
